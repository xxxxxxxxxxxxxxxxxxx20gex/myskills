# PPT Implementation Defaults

Use this file when editing PPTX directly or giving concrete layout instructions.

## 16:9 Canvas

- Standard canvas: 13.333 x 7.5 inches.
- Safe margin: 0.35-0.5 inches.
- Header height: 0.8-1.2 inches.
- Main body height: 5.4-6.0 inches.
- Footer or conclusion strip: 0.35-0.6 inches.

## Typography

- Main title: 30-40 pt.
- Subtitle: 14-18 pt.
- Section title: 16-22 pt.
- Module title: 14-20 pt.
- Body text: 10.5-14 pt.
- Dense table text: 8.5-11 pt.
- Caption: 7.5-9 pt.

For Chinese technical decks, prefer Microsoft YaHei, DengXian, Source Han Sans, or a deck-provided Chinese font. Keep font family consistent unless the existing deck has a clear system.

## Density

- Use one main visual object per slide.
- Keep 3-6 primary modules.
- Keep 2-5 bullets per module.
- Avoid body paragraphs longer than two lines.
- If a page needs more than 6 modules, group them into lanes, clusters, or phases.

## Shape Rules

- Repeated shapes should share width, height, border, and padding.
- Align to a consistent grid.
- Use thin borders, usually 0.5-1.25 pt.
- Use accent color only for active node, output, risk, proof, or conclusion.
- Avoid nested cards unless the inner card has a real function such as status, metric, or warning.

## Dark Background

- Body text must be bright enough for projection.
- Use semi-transparent panels only when contrast remains high.
- Do not place small text directly on busy background imagery.
- Use dim background images behind an overlay if the content is primary.

## Verification

After editing PPTX:
- Render screenshots or export pages when possible.
- Check text overflow, cropped labels, inconsistent alignment, and unreadable small text.
- Check that arrows encode real relationships.
- Check that the slide still has one clear thesis.
